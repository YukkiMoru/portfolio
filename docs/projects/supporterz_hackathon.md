**フロントエンド開発**を担当し、ユーザーが迷わずに操作できるシンプルなUIを設計・実装しました。
JavaScriptによるDOM操作や、バックエンドAPIとの非同期通信周りを主に実装しました。

| **使用技術** | Next.js, TypeScript, Tailwind CSS (shadcn/ui), FastAPI, MariaDB, Docker |

### プロジェクト概要と担当業務
学生のポートフォリオと企業の求人をマッチングさせるSNSプラットフォームです。
モダンなWeb開発技術スタックを採用し、**Next.js + FastAPI** という構成で開発しました。
UIコンポーネントライブラリ **shadcn/ui** を活用し、短期間で高品質なデザインとレスポンシブ対応を実現しました。
特に、ユーザー体験を損なわないスムーズな画面遷移と、インタラクティブなUI実装に注力しました。
